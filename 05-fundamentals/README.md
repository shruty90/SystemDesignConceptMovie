# Fundamentals

Concept questions asked directly in interviews, and the building blocks every design problem reuses.

## Frequently asked concept questions

- API Gateway vs Load Balancer
- Forward proxy vs Reverse proxy
- Horizontal vs Vertical scaling
- Microservices vs Monolith
- Horizontal vs Vertical partitioning (sharding)
- SQL vs NoSQL — when and why
- CAP theorem and consistency models
- How does Kafka work? Kafka vs RabbitMQ vs ActiveMQ
- JWT vs OAuth vs SAML; how SSO works
- Caching strategies (cache-aside, write-through, write-back) and eviction (LRU/LFU)
- Consistent hashing
- CDN internals
